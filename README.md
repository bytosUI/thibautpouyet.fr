# thibautpouyet.fr

Site personnel de **Thibaut Pouyet** — UX/UI Designer.

- **Landing page** : [`index.html`](./index.html) — layout bento minimaliste, single viewport
- **Design system** : [`design-system.html`](./design-system.html) — `bytosUI`, 23 composants en single-file HTML

## Stack

- HTML statique (aucun build)
- [Tailwind CSS](https://tailwindcss.com/) via CDN
- [Heroicons](https://heroicons.com/) via [iconify-icon](https://iconify.design/docs/iconify-icon/)
- Typographie : [Google Sans Flex](https://fonts.google.com/specimen/Google+Sans+Flex)
- Tokens CSS dans `:root` — primaire `#092DE6`, secondaire `#0F0F11`, tertiaire `#FF5B60`

## Déploiement

Hébergé sur [Hostinger](https://www.hostinger.com/) avec auto-deploy depuis ce repo (`main` → `public_html/`).
